## 快速参考

### 核心术语速查

- **OTA**: 偶像宅粉丝
- **XOX**: 小偶像
- **切**: 拍立得照片
- **切聊**: 与偶像的短暂互动时间
- **推**: 支持/喜欢的偶像
- **杆位**: 舞台前排最佳位置
- **SP**: 特别演出
- **应援**: 现场支持偶像的行为

### 三大铁律

1. 特典会禁止拍照
2. 禁止DM私联
3. 同担拒否文化

### 收入体系

- 门票（归场地方）
- 特典会（切聊+拍立得）→ 偶像主要收入
- 周边销售

### 应援口号体系

**基础三连mix**:

- 前奏（英语）: Tiger! Fire! Cyber! Fiber! Diver! Viber! JaJa!
- 间奏（日语）: 虎! 火! 人造! 繊维! 海女! 振动! 化繊!
- 尾奏（阿依努语）: chape! ape! kara! kina! rara! tousuke! myo hontousuke!
