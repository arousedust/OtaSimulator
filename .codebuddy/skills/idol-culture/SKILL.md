# 国内地下偶像文化与黑话理解

帮助大模型理解国内地下偶像（地偶）亚文化中的专有黑话、术语体系、粉丝文化、现场应援规则、特典会流程等。

## 核心概念

### 基础术语

- **地下偶像 / 地偶**: 指未通过主流媒体出道、无大型经纪公司推动，主要在小型剧场、Livehouse、商场活动空间演出的偶像团体
- **OTA / 偶像宅**: 由日语オタク（otaku）简化而来，指地偶粉丝群体；日文圈常写作「ヲタク」
- **XOX**: 「小偶像」的拼音首字母缩写，粉丝对地偶成员的称呼
- **STF**: 英语 staff 缩写，指地偶团体的工作人员/运营/经纪
- **为爱发电**: 指地偶成员不以收入为主要目的进行演出。绝大多数兼职地偶月收入不足2000元
- **地偶元年**: 指2023年，国产地下偶像团体在这一年从约50支爆发到130+支
- **偶像失格**: 指偶像违反职业规范的行为，典型如私联粉丝、恋爱曝光等

### 城市分布

- **百京 / 沪 / 广 / 川渝**: 地偶圈对北京/上海/广州/重庆成都的圈内戏称
- 上海被称为「中国地偶第一城」，地偶团体质量高，甚至经常有日本偶像团体的演出
- 其他像西安、郑州、天津等城市也有地下偶像团体

## 粉丝分类体系

### 按推的对象分类

- **单推**: 只推一个固定成员
- **DD**: 日语「誰でも大好き」缩写，字面意思「谁都喜欢」，指不固定只推一个人，多推或全团推的粉丝
- **团推 / 箱推**: 喜欢整个团体所有成员
- **改推**: 中途更换主推成员
- **转推 / 推し増し**: 在原有主推基础上新增喜欢的成员
- **同担**: 推同一位成员的其他粉丝
- **同担拒否**: 拒绝与同担接触、合影、互动，部分粉丝在微博简介中注明

### 按行为分类

- **嘎比 / ガチ恋**: 由日语 gachi 引申，指把地偶成员当作真实恋爱对象的粉丝，投入情感强度极高
- **龟**: 指那些即使偶像有让人不舒服的行为，或有盐的对应，但仍继续支持偶像的粉丝
- **炮哥/炮姐**: 演出时使用专业相机拍摄成员的粉丝
- **大爹 / 老头**: 指在为该成员花费大量经济的粉丝
- **TO**: Top Ota的简称，指某个偶像的大粉丝，有一定影响力
- **婆罗门**: 指部分摆资历，对舞台演出等进行居高临下地评价的粉丝
- **初見 / 新规**: 第一次来看演出的新粉丝
- **跑路**: 指因为各种原因不再推某个偶像
- **爆米 / 爆金币**: 指在为该成员花费经济，购买特典券周边的行为

## 活动术语

### 演出类型

- **偶活**: 「偶像活动」缩写，指一切地偶相关演出活动
- **OM / Oneman**: 仅一个团体的单独演出（单团公演）
- **拼盘**: 由多个团体共同参加的联合演出，是地偶最常见的演出形式，票价通常68-128元，时长3-8小时不等
- **无料场**: 门票免费的偶活
- **路演**: 在商场中庭、广场等公开场所进行的免费演出
- **远征**: 团体或粉丝跨城市参加演出；粉丝长途追演也叫远征，部分团提供远征福利
- **SP**: Special 的缩写，指特别演出环节，常见类型有「生日SP」「毕业SP」
- **披露 / 初披露**: 某位成员或某团体第一次正式出现在公众视野/第一次演出
- **毕业**: 成员退出地偶团体/行业的说法，源自日本偶像圈术语
- **MC**: 演出间隙的主持/闲聊环节，成员会介绍自己、念段子、与粉丝互动
- **セーノー / se no**: 成员自我介绍时喊出的节奏语，随后粉丝齐声喊出成员名字，是地偶经典互动仪式

## 票务术语

- **提前票**: 演出前提前发售的票，价格最低，通常停止销售日当天
- **当日票**: 演出当天现场购票，价格较高
- **女性票**: 部分演出面向女性观众的优惠票
- **指名票 / 指名**: 观众可指定自己因哪个团体/成员而购票，主办方用来统计人气，大拼盘演出中常见
- **VIP票**: 通常包含提前入场、全员合影、手渡拍立得等高级福利
- **杆位**: 舞台正前方最前排栏杆位置，是粉丝争夺的最佳观演/拍摄位
- **0号位**: 杆位最中间的位置，通常观演效果最佳
- **手环**: 入场时领取的出入场凭证，多数场地可无限次进出
- **1D**: 附赠的饮料券（1 Drink）

## 特典会术语

### 核心概念

- **切**: 即拍立得照片，或称「切奇」，是地偶特典会的核心商品
- **拍切**: 日语チェキ（cheki）的中文译法，拍立得合影
- **切聊**: 购买特典券后，与小偶像进行1-2分钟的聊天时间，期间偶像通常需要在拍立得上画画/写字
- **画切**: 在拍立得正面作画
- **背切**: 在拍立得背面留言/画画
- **宿题切**: 「宿题」即日语作业，指成员花较长时间精心制作的精致拍立得，通常在生日、情人节等特殊节点发售
- **主题切**: 与宿题切类似，制作不如宿题切精致，可能只含有简单签名和会话
- **团切**: 与整个团体所有成员合影的拍立得
- **反切 / 返切**: 粉丝在社交平台（通常是微博）发出自己拿到的拍立得，@成员互动
- **切奇饭**: 拍立得照片和食物一起拍摄的返图
- **电切**: 通过手机视频通话进行的远程切聊，拍立得可选择手渡或邮寄
- **手渡**: 当面交付拍立得/周边
- **代切**: 通过其他OTA的手机进行语音切聊
- **pld**: 拍立得的拼音缩写

### 特典会类型

- **前特**: 特典会在 live 开始之前举行
- **平特**: 特典会穿插在其他团体演出期间
- **终特**: 特典会在 live 全部结束之后举行（最常见形式）
- **整理券**: 代表切聊顺序的券，免费领取，交给STF
- **特典券 / 切聊券**: 付费购买，代表可切聊的时间和拍立得数量，交给成员
- **带签券 / 有签券**: 包含聊天 + 带画切的拍立得，是最常见类型
- **无签券**: 只含聊天不含画切的券，并非每个团都有
- **手合券**: 可以用粉丝的手机与偶像合影
- **加时券**: 可叠加延长切聊时间
- **关门**: 最后一个切的粉丝，通常切数不限，但是默认为当场张数最多的粉丝，ota之间可能有「竞关门」的情况
- **off会 / オフ会**: 没有live表演的粉丝见面会形式，可能含聚餐、游戏等其他活动
- **出禁**: 禁止某粉丝参加该团后续偶活，通常因粉丝越界行为（如骚扰）被STF处理

## 社交媒体术语

- **DM**: 私信，通常指微博私信。多数地偶团体禁DM，粉丝需通过评论区/@方式互动
- **版聊**: 在微博评论区公开发聊天内容，或发帖@成员公开交流
- **SL / 私联**: 私下联系，偶像与粉丝私加联系方式被视为偶像失格行为
- **超话**: 粉丝在微博上围绕成员/团体建立的话题社区，返切通常可以发但有不成文规范
- **揭示板/板子/板**: 微博账号「地下偶像相关揭示板」，一个国内地下偶像汇总账号
- **投板**: 指向「地下偶像相关揭示板」账号投稿内容，通常是偶像的负面消息
- **甜**: 指偶像给粉丝回复的内容非常甜蜜用心，提供很高的情绪价值
- **盐**: 与甜相对，指偶像给粉丝回复的内容可能让人产生不适，甚至不回复粉丝的行为
- **钓**: 指偶像在特典会或是微博互动中，希望该粉丝持续推自己的发言或行为
- **营业**: 指偶像在微博上发布信息，与粉丝互动的行为
- **恋营**: 指偶像与粉丝像恋爱关系一样在微博或是特典会互动，部分粉丝很喜欢这种营业方式，容易产生gachi
- **无圈/无圈回**: 指粉丝在微博上没有@偶像，偶像也回复了粉丝的评论的行为

## 应援文化

### 应援基础

- **应援**: 支持宣传偶像行为的统称，包括现场打call、mix等，或者为偶像制作周边、易拉宝、小的周边产品等
- **打call**: 粉丝按照节奏挥动荧光棒、喊出应援口号的手部动作体系
- **mix / 喊call**: 应援口号的统称，粉丝在歌曲前奏、间奏、尾奏等固定位置齐声喊出

### 应援口号详解

#### 基础三连mix

最基础的应援口号体系，由英语mix + 日语mix + 阿依努语mix组成：

- **英语mix（前奏）**: Tiger! Fire! Cyber! Fiber! Diver! Viber! JaJa! （加拍「飞ばせ!」/「ファイボ!ワイバ!」）
- **日语mix（间奏）**: 虎! 火! 人造! 繊维! 海女! 振动! 化繊! （加拍「飛べ! 除去!」）
- **阿依努语mix（尾奏）**: chape! ape! kara! kina! rara! tousuke! myo hontousuke!

#### 其他应援类型

- **可变三连mix**: 比基础三连更复杂的进阶版，为八小节长句，需要专门记忆，被视为OTA入门门槛
- **ガチ恋口上mix / 恋口上**: 「虽然你并不只属于我，但是只属于你的我会一直在这里」——表达嘎比粉丝执念情感的经典应援口号体系
- **混沌mix**: 在大段落前奏/落ちサビ处发动，节奏较慢，常伴随蛙叫声式的即兴喊法
- **意味不アイアイ / 阿姨压一压**: 在副歌中喊出的节奏型口号
- **woo hey**: 三连mix发动前的预热口号，配合鼓掌和「jia~一裤走!」

### 互动行为

- **起飞**: 气氛高潮时，几名粉丝合力将一名粉丝抬上肩膀冲向舞台，与偶像击掌互动
- **跳水**: 在演出中从杆位跳下，由台下粉丝接住的行为，一般为粉丝发起，部分偶像也可能参与进来
- **冲浪**: 由台下粉丝托起一名粉丝，使该粉丝在高处滚动的行为
- **饭撒 / fan service**: 偶像在台上向特定粉丝投送的眼神、微笑、飞吻、比心等互动信号
- **马赛/马**: 指在演出中，部分粉丝杆位或是后排，在偶像唱词的部分高高跳起，让偶像看到自己的应援行为
- **mosh**: 指在演出中，在台下参与者相互用手推或身体撞击彼此的行为，现场互动的一种形式，有概率误伤他人
- **开圈**: 指在演出中准备激烈互动(如mosh)时清出一片圆形的区域的行为
- **二步/2step**: 粉丝在台下根据歌曲节奏以特定的步伐进行舞动的应援行为

### 应援术语

- **オタ芸 / Wota艺**: 日本偶像宅应援动作体系的总称，通过动作来表达对偶像的支持和互动
- **自由区**: 部分场地设置的应援区，供粉丝在演出中自由互动，通常是场地中间位置
- **安静观演区**: 部分场地设置的非应援区，供不喜被打扰的观众观演，通常是场地两边的位置
- **女性观演区**: 部分场地专门设置的女性观众专属区域
- **最前管理**: 指部分粉丝在演出中，长期占据杆位，不让给其他粉丝的行为，一般被人不齿
- **厄介**: 指在演出中部分不当的引起偶像和粉丝不适的应援行为

## 周边术语

- **无料**: 粉丝或偶像自己制作的小型周边，免费发放给粉丝。如：徽章、钥匙扣等
- **荧光棒**: 粉丝现场应援必备道具
- **打歌服**: 偶像演出时穿的服装，各团风格差异大：糖果色蓬蓬裙、女仆装、水手服、新娘婚纱、睡衣等
- ***T**: 指某个偶像或是团体的应援T恤

## 行业规范

### 铁律

- **特典会禁止拍照**: 特典会期间严格禁止粉丝私自拍摄，是地偶圈第一铁律
- **禁DM禁私联**: 粉丝通过私信联系成员通常被禁，私加联系方式（微信/QQ等）属严重违规

### 潜规则

- **反切别发超话**: 粉丝拿到拍立得后可以发个人微博但通常不发团体超话，避免刷屏或引发同担不适
- **偶像只能评论不能转发**: 偶像回复粉丝返图/返视频时，行业惯例是只能评论不能转发，保持公平，但无硬性要求

## 代表团体

- **StarWink**: 上海代表女团之一，早期地偶团体
- **采梦YUMETORI**: 上海女子团体，情人节婚纱专场演出曾受到较多媒体报道
- **花与心事Affloret**: 上海女子团体
- **透明教室与平行女孩 / 透团**: 长沙成立的早期国产地偶团，2019年出道，被圈内视为国产地偶拓荒者之一
- **CKS**: 福建厦门/完州地区代表团体，舞台风格以睡衣风等搞怪路线著称
- **心动AfterSchool**: 上海女团
- **Clover / 四叶草**: 广州首个男地偶团体，2024年7月出道，全国分京沪广三站活动
- **All-light / Unlock**: 其他较活跃的男地偶团体

## 行业背景

- **主要收入来源**: 特典会（切聊+拍立得），门票收入通常归主办方/场地方
- **主流票价**: 68-128元（提前票），200元内（VIP票）
- **特典券单价**: 30-80元不等，含1-2分钟切聊+1张拍立得
- **成员身份**: 多数为兼职，以学生为主
- **每周演出频次**: 基本每个周末有1-2场演出
- **主要平台**: 新浪微博（宣传、互动主阵地）、秀动APP（票务）
- **情感经济**: 核心商业模式是把「被看见」「被记住名字」包装为可售卖的情感商品，粉丝在特典会上重复购买同一位成员的切聊，本质是情感投资
- **内卷与流动**: 团体生灭速度极快，被圈内形容为「樱花凋谢式」——大批新团涌现，偶像的生命周期很短

## 大模型使用指南

1. **日语词汇处理**: 地偶文化大量词汇来自日语片假名音译或罗马音简写，遇到陌生词时优先按日语源词展开联想

2. **「切」是核心字**: 围绕它派生出切聊、画切、背切、反切、宿题切、团切、电切、代切、关门等一系列词汇

3. **「推」体系**: 单推、神推、DD、团推、改推、转推、同担——这是粉丝身份识别系统

4. **「特典券」体系**: 整理券+特典券分离，是排队秩序+付费时长分离的行业标准

5. **应援差异**: 现场应援（打call+mix）与K-pop应援不同：地偶mix有固定日语/英语/阿依努语词表，不是中文喊成员名字即可

6. **说话风格**: 写作游戏NPC台词或粉丝对话时，可自由混合中文+日语罗马音单词（如「推し」「ガチ」「セーノー」「ちょっと待って」），这是真实OTA的说话方式

7. **三条红线**: 特典会禁拍照、禁DM、禁私联是三条红线，任何违规角色设定都要慎重

8. **性别差异**: 男地偶团体（如Clover）与女地偶团体的粉丝性别画像不同——女地偶以男性OTA为主，男地偶以女性OTA为主，在设定角色时需要区分
